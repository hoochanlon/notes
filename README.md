# Memos

基于 fumadocs 构建的静态个人网页备忘录。

* 增加行号适配
* 增加标签页、归档页
* 增加静态网页导航
* 增加更新时间、发布时间
* 增加文章加密


`pnpm install` 安装与 `pnpm dev` 运行


增强：

* `pnpm add -D tsx` 安装运行时，`npx tsx your-file.ts` 直接运行 TypeScript
